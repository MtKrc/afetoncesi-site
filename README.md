# afetoncesi.com (MVP)

Bu klasör sade bir statik site MVP’sidir (HTML/CSS/JS). Her sayfa resmi kaynaklara link verir ve yazdırılabilir kontrol listeleri içerir.

## Lokal çalıştırma
- Klasörü açın ve `index.html` dosyasını tarayıcıda açın.
- Daha iyi test için basit bir local server kullanın:
  - `python3 -m http.server 8080`

## Yayınlama (en basit)
- Netlify / Vercel / GitHub Pages gibi bir statik hosting’e yükleyin.
- Domain: afetoncesi.com

## İçerik güncelleme prensibi
- Her sayfanın altındaki “Resmi kaynaklar” listesine sadık kalın.
- Güncelleme yaptığınızda footer’daki “Son güncelleme” tarihini güncelleyin.
